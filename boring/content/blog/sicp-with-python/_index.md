# SICP Python版：计算机程序的构造和解释

欢迎来到 **SICP Python版** 系列教程。

本系列教程基于加州大学伯克利分校（UC Berkeley）经典的计算机科学入门课程教材 [Composing Programs](https://www.composingprograms.com/) 编写。

## 为什么学习这个系列？

《计算机程序的构造和解释》（Structure and Interpretation of Computer Programs，简称 SICP）是计算机科学领域的圣经级教材。原书使用 Scheme（Lisp 的一种方言）作为教学语言，虽然思想深刻，但对初学者来说门槛较高。

**Composing Programs** 继承了 SICP 的核心思想——**抽象**、**编程范式**以及**管理复杂系统**的技术，但改用目前最流行的 **Python 3** 语言进行阐述。这使得原本深奥的计算机科学概念变得更加亲切和易于上手。

## 教程内容规划

本系列将涵盖以下核心主题：

1.  **函数抽象 (Building Abstractions with Functions)**
    *   从零开始：编程的基本元素
    *   函数定义与设计
    *   控制流与高阶函数
    *   递归的艺术

2.  **数据抽象 (Building Abstractions with Data)**
    *   序列与可变数据
    *   面向对象编程 (OOP)
    *   对象抽象与效率

3.  **解释计算机程序 (Interpreting Computer Programs)**
    *   函数式编程
    *   异常处理
    *   解释器的构造

4.  **数据处理 (Data Processing)**
    *   声明式编程
    *   逻辑编程与分布式计算

## 学习建议

*   **动手实践**：每一章都会包含实际的代码案例，请务必在你的 Python 环境中运行并修改这些代码。
*   **思考本质**：不要只关注 Python 的语法，更要理解代码背后的“计算思想”和“抽象模式”。

让我们开始这段探索计算机科学本质的旅程吧！
